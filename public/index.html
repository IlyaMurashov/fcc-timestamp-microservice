A simple timestamp microservice

A freeCodeCamp project by Ilya Murashov

This service allows simple conversion between UNIX and human readable time formats.

Usage example

https://[path_to_the_service]/December 15, 2015

or

https://[path_to_the_service]/1450137600

Response example

In both query scenarios you should the the output in the following format

{
    "unix": 1450134000000,
    "natural": "Tue Dec 15 2015"
}
